# Phanteks-217E

Design resources for Phanteks 217E

## Front Panel

A front panel supporting USB-A, USB-C, audio at the same time.

> Note: This has to be used with other accessories. Contact @elethom to purchase.

![Front Panel](/f-panel.PNG)

## GPU Side Panel

GPU side panel for better cooling. It contains two layers of laser cut acrylic, each 2 mm thick, preferably black. And put a layer of dust filter in between.

![Side Panel Base](/gpu-panel-base.PNG)

![Side Panel Top](/gpu-panel-top.PNG)
